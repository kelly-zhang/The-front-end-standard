#新一站官网前端编码规范

> 规范目的：提高团队协作效率，利于前端后期维护，输出高质量代码。

##HTML编码规则
1. <!DOCTYPE html>。
2. UTF-8。```<meta charest="utf-8">```
3. 标签闭合。```<div></div>,<br>单标签不闭合```
4. 标签使用符合语义化。每个标签元素都有符合的意义和语境。```div使用于块级结构，span行内元素，a超链接，p定义段落，audio定义声音，b定义粗体等等```
5. 标签嵌套正确，`<a><div></div></a>`错误；`<a><span></span></a>`正确
6. 结构，样式，行为分离。
7. 头部引用样式，底部引用脚本。
8. 代码格式化。```块级元素新起一行并缩进```
9. 可通过W3C验证你的代码

3. 编写规范
